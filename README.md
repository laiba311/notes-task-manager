# notes-task-manager
A Flutter-based Notes &amp; Task Management App with Hive local database, note organization, priority management, reminders, due dates, and offline data persistence.

## Features

- Add, Edit, and Delete Notes
- Hive Local Database Storage
- Offline Data Persistence
- Mark Notes as Complete or Pending
- Priority-Based Task Management
- Category-Based Organization
- Due Date Management
- Reminder Support
- Notes Statistics Tracking
- Sort Notes by Priority and Due Date
- Modern and Responsive UI

## Technologies

- Flutter
- Dart
- Hive Database
- Hive Flutter
- Material Design

## Key Functionalities

- Manage notes and tasks efficiently
- Store data locally using Hive
- Track task completion status
- Organize notes with categories and priorities
- Set due dates and reminders
- Access notes offline without internet
